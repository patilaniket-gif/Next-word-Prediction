# 📌 Next Word Prediction using RNN

## 📖 Description
This project builds a Natural Language Processing (NLP) model to predict the next word in a sequence using a Recurrent Neural Network (RNN). The model learns patterns and relationships in sequential text data to generate the most probable next word based on the given input.

RNN is well-suited for sequence prediction tasks as it retains information from previous inputs, making it effective for language modeling and text generation.

## 🎯 Objective
- Predict the next word in a sentence
- Learn sequential patterns in text data
- Build a text generation system using deep learning

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy, Pandas
- NLP techniques (Tokenization, Padding)

## ⚙️ Features
- Predicts next word based on input sequence
- Learns context from previous words
- Can generate text sequences
- Extendable to chatbots and autocomplete systems
